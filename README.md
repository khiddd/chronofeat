## ğŸ‰ Features

chronofeat comes packed with several features for easy time-series analysis:

- **Lags Creation**: Generate lagged variables to help with seasonal and trend analysis.
- **Moving Averages**: Easily create moving averages to smooth your data and identify trends.
- **Rolling Statistics**: Calculate rolling statistics like mean and standard deviation for better time-series insights.
- **Calendar Features**: Automatically add features based on calendars, like days of the week or holidays.
- **Model Agnostic**: Use chronofeat without being tied to a specific modeling approach.
- **Recursive Multi-Step Prediction**: Predict multiple time points ahead, useful for various forecasting needs.
